### Backend

- Express
- MongoDB
- Nodemon

### Tools
- Nodenv

## Installation
### 1. Clone the respository
<pre>
git clone https://github.com/DrWhiskas/lateflix.git
cd lateflix
</pre>
### 2. Install dependencies
<pre>
npm install
</pre>
### 3. Start the application
<pre>
npm start
</pre>
